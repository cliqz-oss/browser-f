(async) execute(cmd)

Executes a Marionette command and sends back a response when it has finished executing.

If the command implementation sends the response itself by calling resp.send(), the response is guaranteed to not be sent twice.

Errors thrown in commands are marshaled and sent back, and if they are not WebDriverError instances, they are additionally propagated and reported to Components.utils.reportError.
